== Teams for WooCommerce Memberships
Author: skyverge
Tags: woocommerce, memberships, teams
Requires at least: 4.6
Tested up to: 5.2
Requires PHP: 5.4

See https://docs.woocommerce.com/document/teams-for-woocommerce-memberships/ for full documentation.

== Installation ==

1. Upload the entire 'woocommerce-memberships-for-teams' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
